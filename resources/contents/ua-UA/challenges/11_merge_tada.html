Очищення

Коли гілка вам більше не потрібна ви можете видалити її локально і веддалено. Це корисно, щоб не закінчити із купую непотрібних старих гілок.

Злиття локально

Ваш запит зтягнення був злитий! Ваша гілка була злита в гілку 'gh-pages' у оригіналі на GitHub. Ви можете злити вашу гілку локально теж.

Спершу перейдіть на гілку, в яку ви хочете злити. В нашому випадку на гілку 'gh-pages'.

git checkout gh-pages

Тепер скажіть Git'у яку гілку ви хочете злити — в нашому випадку, ваша гілка, що починається на "add-".

git merge <BRANCHNAME>

Приберіть шляхом видалення вашої гілки. Після її злиття вона більше не потрібна.

git branch -d <BRANCHNAME>

Також ви можете видалити її із вашого віддаленого репозиторію на GitHub:

git push <REMOTENAME> --delete <BRANCHNAME>

Зтягнення із Upstream

І останнє, після вашого запиту зтягнення оригінал змінився — Reporobot додва ваше ім'я до веб-сайту! Якщо ви зтягнете ці зміни із upstream ваша локальна версія стане актуальною. Ви зможете так само побачити його за адресою: yourusername.github.io/patchwork.

Щоб зтягнути зміни із оригіналу:

git pull upstream gh-pages

Вітаємо!

Ви створили локальні репозиторії, віддалені репозиторії, попрацювали із співавтором, надслали зміни, зтягнули зміни і долучилися до мільйонів інших розробників у розвитку відкритого коду!
