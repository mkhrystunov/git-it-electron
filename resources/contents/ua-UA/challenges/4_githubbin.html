Створіть GitHub акаунт

Перейдіть за посиланням github.com і зареєструйте безкоштовний акаунт. Висока п'ять, ласкаво просимо!

Додайте ім'я користувача GitHub до Git

Додайте ваше ім'я користувача GitHub до вашої конфігурації Git. Ми це робимо лише задля Git-it; це робить легшим перевірку наступних завдань. Збередіть його точно таким, як на GitHub і великими буквами, там де вони великі. Зверніть увагу, що непотрібно вводити "<" та ">".

Додайте ваше ім'я користувача GitHub до вашої конфігурації Git:

git config --global user.username <USerNamE>

Ви можете перевірити, що саме ви зберегли у вашій конфігурації, набравши:

git config --global user.username
{{{ verify_button }}}

Переконайтеся, що ви набрали "user.username" вище, а не "user.name", що перезапише ім'я, яке ви налаштували у першому завданні і залишить вас без імені користувача! Якщо ви дізналися, що зробили саме так, начого страшного, лише повторіть крок із першого завдання аби налаштувати ваше ім'я, а потім завершіть крок згори

Різниця між ім'ям користувача на GitHub і в налаштуваннях Git

Розповсюдженна помилка коли ім'я користувача на GitHub не такого ж регістру як у git config. Наприклад, 'JLord' це не те ж саме, що й 'jlord'

Щоб змінити ім'я користувача, що ви встановили у Git, просто запустіть ту ж команду, що й перед цим, але із правильним регістром:

git config --global user.username <USerNamE>

Коли зробите ці зміни, перевірте завдання знову!
