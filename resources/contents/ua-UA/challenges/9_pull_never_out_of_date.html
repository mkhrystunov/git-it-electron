Тримайте свій файл в актуальному стані, витягуючи зміни від співавторів.

Виятгування із віддаленого репозиторію

Якщо ви працюєте на чимось із кимось ще, вам буде потрібно підтримувати актуальний стан репозиторію. Тому ви захочете pull (зтягнути) будь-які зміни, які могли бути надіслані до центрального GitHub репозиторію.

Що зробив Reporobot?

Перевірте чи Reporobot зробив будь-які зміни до вашої гілки, зтягнувгши будь-які зміні із віддаленого з'єднання 'origin' на GitHub:

git pull <REMOTENAME> <BRANCHNAME>

Якщо нічого не змінилося, воно вам повідомить 'Already up-to-date'. Якщо ж там були будь-які зміни, Git зіллє ці зміни із вашою локальною версією.

Reporobot зробив якісь зміни? Git скаже де саме були зроблені зміни. Ви можете відкрити цей файл і побачити зміни Reporobot'а. Сюрприз, Reporobot — артист!
