Git

Git — це безкоштовне відкрите програмне забезпечення (код доступний для будь-кого) написане Лінусом Торвальдсом, який також написав операційну систему Linux.

Git — це програма для відстеження змін із плином часу, відомі в програмуванні як контроль версій. Якщо ви використувували інструмент відслідковування змін в текстових редакторах, то ви вже знайомі із концептом!

Встановлення Git

Ми рекомендуємо встановити Git на вашому комп'ютері, завантаживши його за посиланням GitHub Desktop. Ми не будемо використовувати графічний інтерфейс в Git-it (ми вчимо термінал!), але Git Desktop включає в себе і звичайний Git і це найпростіший шлях встановити Git на всіх платформах однаково.

  • Windows: Використовуйте Git Shell для вашого терміналу.
  • Mac: Відкрийте GitHub Desktop і в Налаштуваннях оберіть встановлення інтсрументів консолі. Використовуйте програму термінал в якості вашої консолі.

Вже маєте встановлений Git або невпевнені? Наберіть git --version у вашому терміналі і якщо отримуєте версію вищу за 1.7.10, то ви готові! Для додаткової інформації відвідайте веб-сайт Git'у.

Програмне забезпечення Git'у

Програма GitHub Desktop здатна робити багато речей із Git'ом, однак не всі, тому вивчення терміналу є дуже важливим. Але як тільки ви це засвоїти, ви будете раді мати Git Desktop, оскільки він організує інформацію вашого проекту візуально, немов на сайті GitHub.

Git сам по собі не схожий на інші програми на вашому комп'ютері. Швидше за все, ви не помітите значок на вашому робочому столі, але він завжди буде доступний для вас і ви зможете отримати до нього доступ в будь-який час з вішого терміналу або із настільного додатку Git

Налаштування Git

Після того як GitHub Desktop (а, отже, і Git) встановлено, відкрийте ваш термінал. Ви можете перевірити, що Git дійсно там набравши:

git --version

Команда поверне версію Git на вашому комп'ютері, що буде виглядати приблизно так:

git version 1.9.1

Потім, налаштуйте Git, щоб він знав як підв'язати вашу роботу саме із вами:

Встановіть ваше ім'я:

git config --global user.name "Your name"

Тепер встановіть вашу електронну адресу:

git config --global user.email "youremail@example.com"

Ви завершили ваше перше завдання! Натисніть кнопку 'Verify' для перевірки завдання.
