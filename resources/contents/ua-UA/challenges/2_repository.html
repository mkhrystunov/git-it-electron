Репозиторії

Репозиторій — це колекція пов'язаних елементів. В нашому випадку, коли пишемо програмне забезпечення, це колекція файлів, пов'язаних із проектом. Ви можете його собі уявити як папку проекту із усіма відповідними файлами всередині нього. Насправді, це те, як він буде виглядати на вашому комп'ютері в будь-якому випадку. Іноді вони називаються репо ("repos") в якості короткого варіанту.

Ви кажете Git'у де ваш проект і він почне відслідковувати всі зміни до цієї папки. Це робить її Git-репозиторієм: папкою із елементами, що відслідковуються Git`ом. Git відслідковує, коли файли додаються, видаються або навіть якщо одна буква у одному файлі змінилася. Це все плюс хто саме вніс зміни і коли відкслідковується Git`ом. В програмування, таке відслідковування змін і називається контролем версії.

Використовуємо Terminal

Terminal (або Bash) це спосіб інструктувати ваш комп'ютер, щоб робити речі, набравши команди, замість клікання в програмах мишею. Ви можете перейменовувати файли, переміщати файли, створювати папки, переміщатися між папками і багато чого іншого, виконуючи набрані команди. Ва навіть можете використовувати текстовий редактор (наприклад, Vim) у вашому терміналі і ніколи його не покидати!

Окрім навігації по комп'ютеру, ви також можете використовувати програми в терміналі, які мають командний інтерфейс (CLI - command line interface), що означає, що вони можуть працювати із командами в терміналі. Git є однією із них. Перша частина команди інформує комп'ютер, що ви "говорите" із Git'ом. Частини після цього це команди та різні опції, згідно із якими ви хочете, щоб діяв Git.

В курсі Git-it ви вивчите декілька базових термінальних команд (в додаток до команд Git'у) для навігації по вашому комп'ютеру; вони описані в кроках.

Створіть репозиторій

Ви створите нову папку проекту, а потім ініціалізуєте її в якості репозиторію Git. Ми назвемо папку "Hello world" в цьому завданні, але ви можете обрати інше ім'я, якщо така папка у вас вже існую (це загальновживана перша назва проекту).

Час відкрити ваш термінал!

У вікні терміналу введіть наступні команди, одну за одною, натискаючи Enter після кожної.

Спершу створіть нову папку:
Підказа: mkdir означає make directory

mkdir hello-world

Потім перейдіть до цієї папки:
Підказка: cd означає change directory

cd hello-world

Нарешті, вкажіть Git'у ініціалізувати (почати відслідковувати) папку, в якій ви зараз:

git init

Остання повинна повернути щось на зразок "Initialized empty Git repository". Інші команди не повертають нічого.

Ви це зробили! Якщо ви хочете бути двічі впевнені, що це Git репозиторій, введіть git status і якщо цу не повернуло 'fatal: Not a git repository...', все круто!
