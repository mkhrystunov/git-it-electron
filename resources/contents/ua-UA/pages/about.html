Про Git-it

Git-it навчає основним, повсякденним командам Git, які розробники використовують щодня, разом із GitHub Flow, шляхом розробники співпрацюють на GitHub.

Деякі завдяння потребують підключення до інтернету, отже ви повинні бути онлайн аби завершити Git-it. Але ви можете завжди відкрити програму і продивитися завдання і ресурси офлайн.

Справжнє Середовище Розробки

Git-it використовує справжній термінал, Git та GitHub (а не їх емуляцію). Коли ви завершите у вас будуть справжні репозиторії на вашому GitHub акаунті і зелені квадратики у вашій діаграмі внесків!

Код, на якому написано Git-it

Ця програма побудована із HTML, CSS та JS і використовує бібліотеку Electron. Увусь код для Git-it є у вільному доступі — репозиторій на GitHub. Git-it це праця любові @jlord та добрих співавторів.
