Нижче ви знайдете перелік консольних команд, що використовуються в завданнях Git-it.

Базові команди Bash/Terminal

  • Стфорити нову папку (директорію)
    • mkdir <FOLDERNAME>
  • Переміститись у вже існуючу папку (змінити директорію)
    • cd <FOLDERNAME>
  • Показати список елементів у папці
    • ls

Налаштування Git

  • Перевірити версію Git
    • git --version
  • Встановити ваше ім'я
    • git config --global user.name "Your Name"
  • Встановити вашу електронну адресу
    • git config --global user.email youremail@example.com
  • Встановити ваш Github акаунт (чутливий до регістру)
    • git config --global user.username uSeRnAmE

Базові команди Git

  • Увімкнути Git для папки
    • git init
  • Перевірити статус змін у репозиторії
    • git status
  • Показати зміни до файлів
    • git diff
  • Додати зміни файлу до коміту
    • git add <FILENAME>
  • Додати всі змінені файли до коміту
    • git add .
  • Зробити коміт (зберегти) змін із коротким описом ваших змін (повідомлення коміту)
    • git commit -m "your commit message"
  • Копіювати репозиторій на ваш комп'ютер
    • git clone <URL>

Git Branches

  • Створити нову гілку (branch)
    • git branch <BRANCHNAME>
  • Перейти на існуючу гілку (branch)
    • git checkout <BRANCHNAME>
  • Ви можете створити нову гілку і перейти на неї однією командою
    • git checkout -b <BRANCHNAME>
  • Показати список усіх гілок
    • git branch
  • Перейменувати гілку, на якій ви зараз знаходитесь
    • git branch -m <NEWBRANCHNAME>

Git Remotes

  • Додати віддалене з'єднання
    • git remote add <REMOTENAME> <URL>
  • Встановити URL для віддаленого з'єднання
    • git remote set-url <REMOTENAME> <URL>
  • Показати список віддалених з'єднань
    • git remote -v

Pulling in Git

  • Витягнути зміни ("зпулити")
    • git pull
  • Витягнути зміни із віддаленої гілки
    • git pull <REMOTENAME> <REMOTEBRANCH>
  • Переглянути зміни перш ніж "зпулити" їх
    • git fetch --dry-run

Push & Merge in Git

  • Надіслати зміни (Push)
    • git push <REMOTENAME> <BRANCHNAME>
  • Злити іншу гілку у вибрану гілку (та, на якій ви зараз)
    • git merge <BRANCHNAME>

Видалення віддалених з'єднань і гілок у Git

  • Видалити локальну гілку
    • git branch -D <BRANCHNAME>
  • Видалити віддалену гілку
    • git push <REMOTENAME> --delete <BRANCHNAME>
