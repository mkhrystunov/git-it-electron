GitHub Desktop

Швидше за все, ви встановили його, коли виконували завдання Git-it, але якщо раптом ні — ось посилання.

Програма GitHub Desktop дозволяє вам робити більшість із того, що можна зробити у терміналі (хоча не все) і вона набагато дружелюбніша до новачків. Особисто, я використовую її аби переглянути певний немалий diff, який в мене є. Вагі зміни набагато приємніше переглядити у такий спосіб.

  • Github Desktop (OSX, Windows) Free by GitHub
  • Github Desktop Guides by GitHub
